# Claude Skills Workbench

A local workspace for developing, testing, and integrating Claude skills and agent workflows.

## Structure

- `skills/` – Custom and imported Claude skills
- `.claude/` – Claude configuration and metadata
- `scripts/` – Utility and runner scripts

## Status

Early development / experimental.
